# Demo sản phẩm

## Nội dung trình diễn

1. Đăng nhập và giới thiệu giao diện dashboard.
2. Tìm kiếm dataset theo từ khóa, lọc theo địa phương.
3. Xem chi tiết dataset và tải file chuẩn hóa.
4. Gọi thử API `/api/v1/datasets` trên Swagger.
5. Trình bày biểu đồ dự báo chỉ số chuyển đổi số (ML heatmap).

> Demo live: https://ldx-insight.vercel.app (Frontend) & https://api.ldx-insight.io/swagger-ui (Backend)
