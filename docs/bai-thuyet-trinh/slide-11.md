# Đánh giá & hướng phát triển

## 3 điểm mạnh nổi bật

1. **Dữ liệu chuẩn hóa tự động** từ ≥3 nguồn, có logging & retry – đáp ứng tiêu chí tích hợp.
2. **Dashboard trực quan + API mở**, UX thân thiện, tài liệu đầy đủ – đáp ứng tiêu chí hiển thị & API.
3. **Kiến trúc rõ ràng**, triển khai cloud hoàn chỉnh, mã nguồn mở – dễ dàng nhân rộng.

## Hạn chế hiện tại

- ML mới ở mức baseline, cần thêm dữ liệu lịch sử dài hơn để tăng độ chính xác.
- Chưa có cơ chế realtime khi nguồn dữ liệu cập nhật đột xuất.

## Hướng phát triển

1. Mở rộng thêm nguồn dữ liệu (Bộ TT&TT, Cổng dữ liệu trung ương) và hỗ trợ cập nhật incremental.
2. Hoàn thiện microservice ML, công bố API phân tích chuyên sâu, bổ sung dashboard cảnh báo.
3. Container hóa toàn bộ hệ thống, xây dựng marketplace API cho cộng đồng GovTech.
